01. Project Goal

홈 화면의 무분별한 팝업 페이지으로 홈페이지보단 쇼핑몰에 가까운 인상을 주고, 메뉴보단 알림 공지사항이 눈에 더 잘들어옴. 또한 통일된 컬러와 폰트가 없어서 브랜드 정체성이 느껴지지 않음.

Problems

  • 브랜드 정체성 모호 통일된 폰트와 컬러가 없음.
  • 카테고리 혼란 메뉴 카테고리 분류가 잘 되어 있지 않아, 사용자에게 혼란을 줌.
  • 올드한 구조 반응형 웹으로 구현이 되나, 전체적 레이아웃이 올드한 느낌을 줌.
  • 웹 정근성 미흡 팝업 페이지가 화면을 뒤덮고 있어 사용자가 원하는 액션을 수행하기 힘듦.
img

Solution

  • 올드한 구조 메뉴의 이미지를 메인으로 배치
  • 카테고리 정리 메뉴 카테고리 재 분류
  • 이벤트 배너 추가 팝업창을 제거하고 미니 배너로 표현
  • 포인트 컬러 사용 포인트 컬러와, 통일된 서체로 브랜드 정체성 확립

03. Design Concept

  • 세븐브로이 로고 색이 화이트와 어두운 블루 컬러를 사용.
    상품 소개 섹션은 캔 맥주 상품 패키지 컬러를 사용함.

Color

  • #e41e26
  • #f05922
  • #f99d1c

font

  • 로고와 어울리고 가독성이 좋은 폰트를 사용함.
  • Noto Sans KR
  • 배달에 민족 주아

04. UX / UI Solution

  • 불필요한 행동 요소 제어 메인 화면에 불필요하게 클릭을 일으키는 요소를 없앰.
  • 메뉴 위치 내비게이션 추가 사용자가 현 메뉴 위치를 파악할 수 있음.
  • 명확한 대비 명확한 대비 컬러를 사용해, 가독성을 좋게 함.

05.Publishing

  • 미디어쿼리를 이용해 모바일과 / PC 화면 반응형 웹 제작
  • CSS 3를 이용해 간단한 색상 반전 호버와 클릭 액티브 효과
  • 텍스트에 애니메이션 플러그인으로 세련된 움직임을 줌.
