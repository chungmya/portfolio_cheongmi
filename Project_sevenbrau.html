01. Project Goal

배경컬러와 텍스트 컬러 대비가 낮아 가독성이 떨어지고, 폰트 사용에 통일성이 없음
메인 상품인 맥주가 돋보이지 않음

Problems

  • 매력적이지 않은 레이아웃 홈페이지 시선 흐름이 흩어지는 짐
  • 가독성 폰트 컬러 대비가 적음
    불필요한 텍스트 수정 필요
  • 메인 상품 부각 필요 상품 이미지 보정 및 레이아웃 변경
  • 브랜드 아이덴티티 확립 밝은 느낌의 디자인 필요
img

Solution

  • 이미지 재배치 화려한 이미지를 상단에 배치하여 주목도를 높임
  • 대비 높은 컬러 사용 텍스트 내용을 정리하고, 대비되는 컬러를 사용해 가독성을 높임
  • 맥주 패키지 컬러 사용 맥주 패키지 컬러와 애니메이션 효과로 화려함을 더함
  • 채도 높은 컬러 사용 밝은 컬러를 사용해 활기찬 느낌을 더함

03. Design Concept

  • 세븐브로이 로고 색이 화이트와 어두운 블루 컬러를 사용.
    상품 소개 섹션은 캔 맥주 상품 패키지 컬러를 사용함.

Color

  • #fff
  • #000
  • #005ad5
  • #d93c21
  • #7752ca
  • #f3cb1e

font

  • 로고와 어울리고 가독성이 좋은 폰트를 사용함.
  • Noto Sans KR
  • Cafe24Dangdanghae

04. UX / UI Solution

  • 레이아웃 정리 시선의 흐름이 자연스럽게 흘러가도록 레이아웃 구성
  • 텍스트 정리 사용자에게 내용이 쉽게 전달되도록 불필요한 텍스트 정리
  • 명확성 대비되는 컬러를 사용해 클릭 버튼이 명확하게 보이게 함

05.Publishing

  • 미디어 쿼리를 이용해 모바일과 / PC 화면 반응형 웹 제작
    CSS animate를 이용하여 맥주 상품 섹션에 동적인 모션을 줌
  • 맥주 상품 소개 섹션에 CSS를 이용해 상품의 주목도를 높임
  • 텍스트에 애니메이션 플러그인으로 세련된 움직임을 줌.
