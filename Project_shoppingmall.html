01. Project Goal

한국의 쇼핑몰 사이트는 상품 설명 페이지가 통 이미지로 제작되어 웹 접근성이 떨어짐.

Problems

  • 올드한 구조 올드한 느낌의 디자인
  • 가독성 폰트 컬러 대비가 적음. 상품이 돋보이지 않는 레이아웃 디자인
  • 반응형 웹 부재 반응형으로 제작
    되지 않음.
  • 웹접근성 미흡 웹 접근성이 미흡한 상품 설명 페이지
img

Solution

  • 포인트 컬러 사용 최대한 컬러를 적게 사용하고, 포인트
    컬러로 주목도를 높임
  • 카테고리 정리 불필요한 텍스트 정리
  • 레이아웃 재 배치 상품의 이미지를
    큰 영역 슬라이드
    배너로 활용
  • 웹표준 · 접근성 준수 웹표준과 웹 접근성 준수하여 반응형 웹 구현

03. Design Concept

  • 상품 주목성을 위해 화이트, 블랙 컬러를 사용함. 포인트 컬러 레드 사용.

Color

  • #fff
  • #000
  • #ed1d24

font

  • 익숙하고 가독성이 좋은 폰트를 사용함.
  • Noto Sans KR
  • Barlow

04. UX / UI Solution

  • 레이아웃 정리 정리된 텍스트와 심플한 레이아웃으로 상품이 한눈에 보이도록 구성.
  • 명확한 대비 대비되는 컬러를 사용해서 텍스트의 가독성을 높임.
  • 카테고리 분류 산발적으로 흩어져있던 메뉴 카테고리를 상단에 고정시켜 사용자의 혼란을 최소화시킴.
  • 클릭 최소화 상품 설명 페이지에서 ‘구매하기’ 버튼으로 갈수 있도록 '위로 가기 버튼'을 추가함. 최소한의 클릭으로 상품구매가 가능함.

05.Publishing

  • 미디어쿼리를 이용해 모바일과 / PC화면 반응형 웹 제작.
  • CSS3를 이용해 메뉴탭 색상 반전, 상품 이미지 호버 효과를 줌.
  • 상품 설명 페이지를 웹 표준에 맞게 제작하여 누구나 접근성이 용이하게 제작.
  • jQuery를 이용해 슬라이드 메인 배너, 서브 배너를 추가함.
  • 최상단 내비게이션과, 사이드 메뉴 탭에 애니메이션 플러그인을 활용해 세련된움직임을 줌.
  • jQuery를 이용해 상품 구매 클릭 시, 완료 알림 창 출력.
